# Flask Fullstack Mastery

Flask 풀스택 강의 노트북 15개를 정적 학습 웹사이트로 재구성한 GitHub Pages용 사이트입니다.

## Pages

이 저장소를 GitHub에 push하면 `.github/workflows/pages.yml` 워크플로가 정적 파일을 GitHub Pages에 배포합니다.

예상 URL:

```text
https://lution88.github.io/flask-fullstack-mastery/
```

## Local Preview

```bash
python -m http.server 8766 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:8766/index.html
```
